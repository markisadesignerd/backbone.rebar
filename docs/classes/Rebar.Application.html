Rebar.Application Class

Uses
  • extend
Extends Backbone.Events
Defined in: ../temp/backbone.rebar.js:33

The application shell provides a simple default architecture consisting of a model, view and controller. The application is a singleton class in that there can only be one. It extend Backbone.Events and you can see the documentation for more detailed information.
